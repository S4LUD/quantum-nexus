# Rules Update Summary (Feb 5, 2026)

This document summarizes recent gameplay rule changes.

## Output Generation
- Each Node provides +1 output of its type.
- Output counts toward Protocol requirements.
- Output also reduces the **matching energy cost** by **1** when building Nodes.
- Applies to **base energies only** (Solar/Hydro/Plasma/Neural).
- Output deductions **stack** with other discounts and Protocol effects.
- Output is **not spent** when claiming Protocols.

## Exchange
- New action: **Exchange Energy**.
- 1:1 exchange (give 1, take 1).
- 2:3 exchange (take 2 of the same type, give any 3).
- Flux **cannot** be exchanged.
- Pool must have enough energy to take.
- Pool cannot exceed max counts when receiving returned energy.

## Flux
- Flux is **not collected from the pool**.
- Flux is gained **only by reserving** Nodes (if available).
